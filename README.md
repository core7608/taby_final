# 🕌 Taby Prayer - مواقيت الصلاة

![Build Status](https://github.com/core7608/taby_final/actions/workflows/main.yml/badge.svg)

تطبيق مواقيت صلاة عصري مصمم بلغة **Swift** وإطار عمل **SwiftUI** معتمدة على تصميم **Glassmorphism** (الزجاج السائل).

## ✨ المميزات
- **مواقيت دقيقة**: عرض مواقيت الصلوات الخمس.
- **خلفية ديناميكية**: تدرجات لونية سائلة تتفاعل مع العناصر.
- **دعم الوضع الداكن**: متوافقة تماماً مع إعدادات النظام.
- **بناء تلقائي**: جاهزة للعمل مع GitHub Actions.

## 🚀 التشغيل
1. افتح المشروع باستخدام **Xcode**.
2. اضغط على `Cmd + R` للتشغيل على المحاكي.
3. للتحميل المباشر: اذهب إلى قسم **Releases** في هذا المستودع وحمل ملف الـ `.zip`.

> **ملاحظة**: النسخة المرفوعة في الـ Releases مخصصة للعمل على محاكي (Simulator).

## 🛠 المتطلبات
- Xcode 15+
- iOS 17.0+
- Swift 5.9+
